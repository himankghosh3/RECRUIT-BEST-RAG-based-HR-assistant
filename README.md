Recruit-Best: AI Hiring Assistant

Recruit-Best is a Retrieval-Augmented Generation (RAG) application that helps recruiters identify the most suitable candidates based on skills, experience, and qualifications.

The system uses semantic search with embeddings and a local LLM (Ollama) to generate intelligent candidate recommendations.

## Tech Stack
- Python
- Streamlit
- Ollama
- Pandas
- NumPy
- Scikit-Learn

## Features
- Candidate dataset preview
- Semantic candidate search
- AI-generated recruitment recommendations

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Start Ollama

ollama serve

3. Run the app

streamlit run app.py
